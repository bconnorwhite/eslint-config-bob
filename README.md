> ESLint configuration for packages built with @bconnorwhite/bob.

## Installation

```bash
yarn add eslint-config-bob
```

```bash
npm install eslint-config-bob
```

## Usage

In .eslintrc.json:

```json
{
  "extends": "eslint-config-bob"
}
```
